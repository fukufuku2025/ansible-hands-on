第5章: Roleの作成

5.2: `httpd`のRoleを作成し、実行確認

作成したRoleを使って、プレイブック内でタスクを実行します。以下のように`httpd`のRoleを呼び出します。

Roleを呼び出すPlaybook


---
- name: Install and start httpd using role
  hosts: test
  become: true
  roles:
    - httpd

このPlaybookを実行することで、`httpd`がインストールされ、サービスが起動します。

実行方法


$ ansible-playbook playbook.yml
