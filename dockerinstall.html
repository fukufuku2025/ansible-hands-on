1.1.2: WSL2のインストール

WSL2は、Windows Subsystem for Linux 2で、Windows上でLinux環境を動作させるための仕組みです。これにより、Linuxの開発ツールをWindows上で使用できます。

WSL2を有効にして、必要なalpine linuxをインストールします。この辺の手順の詳細はAIとかに聞いてください

1.1.4: Dockerのインストール (Alpine Linuxベース)

Dockerを使用してコンテナを簡単に構築・管理できます。今回はAlpine Linuxをベースにした軽量なイメージを使用します。

Alpine LinuxにDockerをインストールするには、以下のコマンドを使用します。


# DNFでDockerをインストール
$ sudo dnf update -y
$ sudo dnf install -y dnf-plugins-core
$ sudo dnf config-manager --add-repo https://download.docker.com/linux/fedora/docker-ce.repo
$ sudo dnf install -y docker-ce docker-ce-cli containerd.io
            
# Dockerサービスを有効化し、起動
$ sudo systemctl enable --now docker
$ sudo systemctl start docker
            
# インストール確認
$ docker --version
